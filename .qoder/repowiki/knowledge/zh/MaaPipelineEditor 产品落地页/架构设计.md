- **框架与渲染**：采用 Astro 静态站点生成器，以 `src/pages/index.astro` 为唯一入口，配合 `LandingShell.astro` 布局组件统一管理 SEO 元数据、结构化数据（JSON-LD）及全局样式。
- **组件架构**：遵循 Astro Islands 模式，静态内容（如 Hero、Footer、Stats）使用 `.astro` 组件以保证零 JS 开销，高交互模块（如 `FeatureExplorer`）通过 `@astrojs/react` 集成 React 组件并采用 `client:idle` 策略进行水合。
- **内容管理**：通过 `src/content/landing.ts` 集中定义导航、功能特性、统计数据等文案常量，实现内容与视图的解耦。
- **样式体系**：集成 Tailwind CSS v4 作为原子化样式引擎，配合自定义字体（IBM Plex Sans, JetBrains Mono）构建视觉系统。

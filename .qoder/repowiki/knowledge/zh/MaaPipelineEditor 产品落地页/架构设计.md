- **框架与渲染**：采用 Astro 静态站点生成（SSG）架构，以 `src/pages/index.astro` 为唯一入口，通过 `LandingShell.astro` 统一处理 SEO、Open Graph 及结构化数据。
- **内容驱动设计**：实现内容与视图分离，所有文案、导航及功能配置集中于 `src/content/landing.ts`，便于维护与多语言扩展。
- **混合组件模型**：遵循 Astro Islands 模式，静态布局使用 `.astro` 文件，高交互模块（如 `FeatureExplorer`）采用 React 并通过 `client:idle` 指令进行水合。
- **样式体系**：集成 Tailwind CSS v4 作为原子化样式引擎，配合自定义全局样式 `global.css` 管理基础排版与动画效果。

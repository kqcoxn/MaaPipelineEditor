## 简介

避让路由工具是 MAA Pipeline Editor 中的一个重要功能模块，专门用于解决流程图中节点连接线相互遮挡的问题。该工具通过智能的路径规划算法，确保节点之间的连接线能够优雅地避开其他节点，提供清晰美观的视觉效果。

**更新** 本次更新重点修复了分组内边避让模式的行为异常问题，确保分组节点不会干扰避让算法的正常计算，同时增强了对分组内相对定位子节点的坐标转换处理，并修复了嵌套组结构的坐标转换问题。

该工具主要包含两个核心功能：
- **边避让算法**：智能计算连接线路径，避免与其他节点发生冲突
- **节点磁吸对齐**：提供节点拖拽时的磁吸对齐功能，提升用户体验

### 避让路径算法实现

避让路径算法是整个系统的核心，实现了复杂的几何计算和路径优化：

#### 核心算法流程

```mermaid
flowchart TD
Start([开始计算避让路径]) --> CheckSelfLoop{检查是否自循环}
CheckSelfLoop --> |是| SelfLoop[计算自循环路径]
CheckSelfLoop --> |否| CheckParallel{检查平行边数量}
CheckParallel --> |>1| ParallelCalc[计算边偏移量]
CheckParallel --> |<=1| DirectCalc[直接计算路径]
ParallelCalc --> DirectCalc
DirectCalc --> LineTest[测试直线连接]
LineTest --> Blocked{是否被阻挡?}
Blocked --> |否| ReturnLine[返回直线路径]
Blocked --> |是| Try45[尝试45度路径]
Try45 --> Test45{45度路径可行?}
Test45 --> |是| Return45[返回45度路径]
Test45 --> |否| Try90[尝试90度直角路径]
Try90 --> Test90{90度路径可行?}
Test90 --> |是| Return90[返回90度路径]
Test90 --> |否| Recurse[递归绕行]
Recurse --> DepthCheck{检查递归深度}
DepthCheck --> |达到最大深度| ReturnFallback[返回备用路径]
DepthCheck --> |未达最大深度| CalculateBypass[计算绕行点]
CalculateBypass --> RecursiveCall[递归调用剩余路径]
RecursiveCall --> CombinePaths[组合完整路径]
CombinePaths --> End([返回最终路径])
ReturnLine --> End
Return45 --> End
Return90 --> End
ReturnFallback --> End
SelfLoop --> End
```

#### 几何计算核心

算法使用了多种几何计算技术：

1. **线段相交检测**：使用向量叉积法判断线段是否相交
2. **矩形碰撞检测**：基于包围盒检测和精确碰撞检测
3. **路径优化**：计算最短绕行路径，避免重复绕行同一节点

## 性能考虑

避让路由工具在设计时充分考虑了性能优化：

### 算法复杂度分析

1. **避让路径算法**：时间复杂度 O(n²)，其中 n 为节点数量
2. **磁吸对齐算法**：时间复杂度 O(n)，n 为候选节点数量
3. **路径渲染**：实时渲染，使用 SVG 路径优化

### 性能优化策略

1. **缓存机制**：避免重复计算相同的路径
2. **视口过滤**：仅对可见节点进行磁吸计算
3. **递归深度限制**：防止无限递归导致的性能问题
4. **增量更新**：只更新发生变化的边和节点

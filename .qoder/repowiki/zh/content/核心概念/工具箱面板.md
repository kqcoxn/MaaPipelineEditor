## 简介
工具箱面板是MaaPipelineEditor中的核心工具集，提供了一系列用于图像识别和坐标测量的实用工具。该面板通过集成多种模态框组件，为用户提供了一个直观、高效的工具操作界面，支持OCR文字识别、模板截图、颜色取点、区域选择等多种功能。

## 工具箱面板结构
工具箱面板采用模块化设计，主要由工具按钮区域和结果预览区域组成。面板通过React组件实现，使用Ant Design UI库进行界面构建。

## 核心工具功能
工具箱面板提供了六种核心工具，每种工具都对应一个特定的模态框组件，用于执行不同的图像处理任务。

### 工具配置
工具箱中的工具通过`TOOLBOX_TOOLS`常量数组进行配置，每个工具对象包含以下属性：

| 属性 | 说明 |
|------|------|
| key | 工具唯一标识符 |
| label | 工具显示名称 |
| icon | 工具图标名称 |
| iconSize | 图标大小 |
| modalType | 对应的模态框类型 |

```mermaid
flowchart TD
A[工具箱面板] --> B[OCR文字识别]
A --> C[模板截图]
A --> D[颜色取点]
A --> E[区域选择]
A --> F[偏移测量]
A --> G[位移差值]
B --> H[OCRModal]
C --> I[TemplateModal]
D --> J[ColorModal]
E --> K[ROIModal]
F --> L[ROIOffsetModal]
G --> M[DeltaModal]
```

## 工具交互流程
工具箱面板的交互流程遵循统一的模式，从工具调用到结果处理形成完整的闭环。

```mermaid
flowchart TD
A[用户点击工具] --> B{检查连接状态}
B --> |未连接| C[显示错误提示]
B --> |已连接| D[打开对应模态框]
D --> E[用户操作模态框]
E --> F[执行具体功能]
F --> G[获取结果]
G --> H[存储结果到工具箱]
H --> I[显示结果预览]
I --> J[提供复制选项]
J --> K[用户复制值或键值对]
```

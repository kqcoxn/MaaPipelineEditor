## 简介
MaaPipelineEditor（MPE）是一个前后端分离的可视化 Pipeline 编辑器，基于 React 19 + TypeScript + Vite 构建前端，使用 Go 语言实现本地桥接服务（LocalBridge），并通过 Wails 将 Go 后端打包为桌面应用（Extremer）。项目提供在线版本与本地一体化应用两种使用形态，支持文件管理、截图工具、流程调试、AI 辅助等功能。

**新增** 项目现已包含独立的着陆页（Landing）模块，采用 Astro + React + TailwindCSS 技术栈构建，提供完整的静态网站解决方案。着陆页的核心功能展示系统 FeatureExplorer 组件已从简单的静态演示升级为功能完整的交互式图片轮播系统。

## 项目结构
项目采用多模块组织方式：
- 前端模块：位于根目录 src，使用 Vite + React 19 + TypeScript 构建
- 后端模块：LocalBridge（Go），提供 WebSocket 服务与 MaaFramework 集成
- 打包模块：Extremer（Go + Wails），将前端静态资源嵌入并打包为桌面应用
- 文档站点：docsite（VitePress），用于文档发布
- 工具脚本：tools（安装脚本）
- **新增** 着陆页模块：Landing（Astro + React + TailwindCSS），独立的静态网站，包含完整的功能展示系统

### FeatureExplorer 组件详解
**更新** FeatureExplorer 是着陆页的核心交互组件，已从静态演示升级为功能完整的交互式图片轮播系统：

- **交互模式**：支持鼠标点击和键盘导航（方向键、Home、End），提供完整的无障碍支持
- **动画效果**：300ms 过渡动画，配合延迟更新确保视觉流畅，包含淡入淡出效果
- **主题系统**：支持四种色调（blue、mint、orange、rose）的样式变体，每种色调都有独特的边框、背景、指示器和徽章样式
- **响应式设计**：自适应不同屏幕尺寸的布局调整，支持移动端和桌面端的最佳体验
- **图片轮播**：支持多张图片的循环播放，包含左右切换按钮和指示器
- **内容展示**：左侧图片区域显示当前功能的演示图片，右侧文本区域显示功能描述、标签和指标
- **状态管理**：使用 React hooks 管理活动索引、显示索引、动画状态和图片索引映射

## 故障排除指南
- 本地服务无法启动：检查 LocalBridge 配置文件路径、MaaFramework 路径与资源路径，查看日志目录。
- WebSocket 连接失败：确认端口分配与防火墙设置，检查前端 URL 参数与配置。
- Wails 环境连接异常：确认 Extremer 已发送 bridge:port 事件，检查启动画面与窗口显示逻辑。
- **新增** 着陆页测试失败：检查静态服务器脚本是否正确创建 dist 目录，验证 Playwright 配置和测试脚本。
- **新增** FeatureExplorer 交互问题：检查图片索引状态管理、动画状态和键盘事件处理。
- 安装脚本问题：Windows 使用 PowerShell 或 CMD 脚本，Linux/macOS 使用 Bash 脚本，确保网络可达与权限正确。

Pipeline 面板是 MPE 与本地文件的接口。

## Pipeline 操作

![](/images/core/Pipeline工具.png)

您可以以下工具管理您的 Pipeline：

- `导入`：从系统粘贴板内读取 Pipeline JSON，自动检测并兼容 v1/v2 协议格式。支持复制、拖拽或使用文件选择器导入，导入的内容将直接覆盖现有 Pipeline
- `编译预览`：在选中节点时会显示此项，将使选中的节点部分编译为 Pipeline 并展示在预览区
- `部分导出`：将选中的节点编译为 Pipeline，并将纯文本 JSON 复制到系统粘贴板
- `全部导出`：将整个文件编译为 Pipeline，并将纯文本 JSON 复制到系统粘贴板

## JSON 预览搜索

JSON 预览面板内置了搜索功能，支持对编译后的 Pipeline JSON 内容进行关键字查找。

**使用方式**：

1. 打开 JSON 预览面板
2. 在面板顶部的搜索框中输入关键字（支持搜索键名和值）
3. 面板会立即切换为高亮文本视图，匹配的内容以黄色背景标记
4. 当前选中的匹配项以橙色背景突出显示

**搜索导航**：

- 搜索框右侧显示匹配计数（如 `1/5`），表示当前第几个匹配 / 总匹配数
- 点击 `↑` `↓` 箭头按钮在匹配项之间跳转，视图会自动滚动到对应位置
- 到达末尾后自动循环回到第一个匹配项
- 清空搜索框后恢复为默认的 JSON 树形视图

::: tip
搜索不区分大小写，输入后有 300ms 防抖延迟以优化性能。
:::

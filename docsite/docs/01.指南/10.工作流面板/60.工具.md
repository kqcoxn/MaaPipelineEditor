::: tip

:::

## 通用工具

![](/images/core/通用工具.png)

通用工具在编辑区左上角,目前支持如下操作:

- `设置`:点击后唤出设置界面
- `AI 对话历史`:查看调用 AI 的记录
- `复制 (Ctrl+C)`:当选中节点后可用,将节点保存至 MPE 粘贴板,支持多选复制
- `粘贴 (Ctrl+V)`:当 MPE 粘贴板内有复制的节点时可用,节点粘贴至编辑区
- `撤销 (Ctrl+Z)`:撤销上一步操作
- `重做 (Ctrl+Y)`:重做已撤销的操作

## 工具箱

工具箱是 MPE 提供的独立工具集合,集成了所有字段快捷工具,无需选中字段即可直接使用。

### 入口位置

点击编辑区左上角的「工具箱」图标即可打开工具箱面板。

### 功能特点

- **独立使用**:无需选中节点或字段,直接点击工具图标即可使用
- **结果预览**:操作完成后,工具箱会显示最后一次的操作结果,方便查看和复制
- **快捷复制**:支持复制值或键值对,更灵活地使用结果
- **兼容字段面板**:字段面板中的快捷入口依然保留,两种方式可以根据使用场景自由选择

### 支持的工具

工具箱包含以下 6 个工具(需要连接本地服务):

- **OCR 文字识别**:识别指定区域的文字内容
- **模板截图**:截取指定区域并保存为模板图片
- **颜色取点**:从截图中提取颜色值(支持 RGB、HSV、GRAY 模式)
- **区域选择**:框选识别区域(ROI)
- **偏移测量**:测量两个区域之间的偏移量
- **位移差值**:计算滚动距离(dx/dy)

::: tip 提示
更多关于各个工具的详细使用说明,请参考 [字段快捷工具](/guide/server/field-tool) 文档。
:::

## 视图工具

![](/images/core/视图工具.png)

### 聚焦透明度

聚焦透明度功能可以帮助您在复杂的节点网络中快速关注重要节点及其关系。

**使用方式**：

1. 在设置面板中启用「聚焦透明度」功能
2. 选中一个或多个节点后，未被选中的节点会自动降低不透明度
3. 可在设置中自由调控不透明度数值（范围：0-100%）
4. 支持随时启用或禁用此功能

聚焦透明度让节点间的关系更加清晰，特别适合在大型 Pipeline 中定位和调试节点。

### 路径高亮

路径高亮功能可以高亮显示从起始节点到结束节点之间的所有可达路径，帮助您快速梳理节点间的执行流程。

**使用方式**：

1. 在设置面板中启用「路径高亮」功能
2. 选择起始节点和结束节点
3. 所有从起始节点到结束节点的可达路径上的节点都会被高亮显示
4. 非路径上的节点会自动降低透明度

此功能特别适用于：

- 追踪特定执行流程
- 验证节点连接的正确性
- 理解复杂分支逻辑
- 优化 Pipeline 结构

### 对齐参考线与磁吸

对齐参考线与磁吸功能可以帮助您在拖拽节点时自动对齐到其他节点的边缘和中心线，保持画布布局整齐。

**启用方式**：设置面板 → 节点磁吸对齐（默认关闭）

**功能特点**：

- **智能对齐**：拖拽节点时，系统会自动检测与其他节点的边缘（左、右、上、下）和中心线的对齐关系
- **参考线显示**：当检测到可对齐位置时，会在画布上显示蓝色虚线参考线，提供直观的视觉引导
- **自动吸附**：节点接近对齐位置时会自动吸附（阈值 5px），帮助精准对齐
- **实时反馈**：参考线随拖拽实时更新，释放节点后自动隐藏

## 排版工具

![](/images/core/排版工具.png)

在选中两个及以上节点时，您可以使用对齐工具：

- `居中对齐`：将节点 `x` 坐标设置一致
- `顶对齐`：将节点 `y` 坐标设置一致
- `底对齐`：将节点 `y+height` 设置一致

在未选中节点时，您可使用自动排版工具：

- `自动布局`：根据节点连接关系，自动给出一个可区分的排列

为了方便切换主题时的间距调整，您可以使用间距调整工具：

- `水平增加/缩减`：调节节点上下间距
- `垂直增加/缩减`：调节节点左右间距

其他：

- `将布局保存为图片`：导出当前画布为 PNG 图片，若有选中节点则仅导出选中的节点，若无选中节点则导出全部节点

## 分享

分享功能位于顶部导航栏，可以帮助您快速分享当前的 Pipeline 配置。

### 使用方式

1. **生成分享链接**：点击顶部导航栏的「分享」按钮
2. **自动复制**：分享链接会自动复制到剪贴板
3. **分享给他人**：将链接发送给其他人
4. **打开分享链接**：接收方打开链接后，会自动创建一个名为「来自分享」的新文件，并加载分享的 Pipeline

分享链接采用 **URL 参数** 方式，适合分享中小型 Pipeline（理论上支持到 1k 个节点，但链接会很长）

::: warning

- 分享链接包含完整的 Pipeline 配置（节点、连接、参数等），请注意隐私安全
- 打开分享链接时会自动创建新文件，不会覆盖现有工作
- 分享的内容不包含本地文件路径等环境相关配置
  :::

## 实时画面预览

连接设备后，MPE 会在编辑区右上角自动显示实时设备屏幕预览窗口，帮助您在编辑 Pipeline 的同时实时查看设备当前画面。

**启用方式**：设置面板 → 实时画面预览（默认启用）

**功能特点**：

- **自动显示**：连接设备后在右上角自动显示实时设备屏幕
- **智能隐藏**：当字段面板、连接面板或 JSON 预览面板打开时自动隐藏，避免遮挡
- **定时同步**：按设定间隔定时获取设备最新画面，默认刷新间隔为 1000ms
- **刷新间隔可调**：可在设置面板中自定义画面刷新间隔（推荐 500-2000ms），值越小越流畅但性能消耗越高
- **性能优化**：浏览器标签页不可见时自动暂停截图请求，减少后台资源消耗

::: tip
实时画面预览需要先连接 LocalBridge 并连接到设备。如果不需要此功能，可在设置面板中关闭以节省性能。
:::

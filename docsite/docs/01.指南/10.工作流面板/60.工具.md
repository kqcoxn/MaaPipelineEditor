工具可以帮助您更快捷的对节点进行操作。

::: tip
对于所有工具，鼠标悬停时会有相关提示。
:::

## 通用工具

![](/images/core/通用工具.png)

通用工具在编辑区左上角，目前支持如下操作：

- `设置`：点击后唤出设置界面
- `AI 对话历史`：查看调用 AI 的记录
- `复制 (Ctrl+C)`：当选中节点后可用，将节点保存至 MPE 粘贴板，支持多选复制
- `粘贴 (Ctrl+V)`：当 MPE 粘贴板内有复制的节点时可用，节点粘贴至编辑区
- `撤销 (Ctrl+Z)`：撤销上一步操作
- `重做 (Ctrl+Y)`：重做已撤销的操作

## 节点模板工具

::: tip
**在新版本中，推进您右键编辑区空白区域呼出节点添加面板进行操作**
:::

节点模板在编辑区左侧，目前支持如下模板：

![](/images/core/节点模板.png)

点击模板后即会添加至视口，若添加时有选中的节点会自动建立 `next` 连接。

新增 Anchor 节点模板，详见 [节点-Anchor 重定向节点](/guide/start/quick-start#Anchor-重定向节点)

## 排版工具

![](/images/core/排版工具.png)

在选中两个及以上节点时，您可以使用对齐工具：

- `居中对齐`：将节点 `x` 坐标设置一致
- `顶对齐`：将节点 `y` 坐标设置一致
- `底对齐`：将节点 `y+height` 设置一致

在未选中节点时，您可使用自动排版工具：

- `自动布局`：根据节点连接关系，自动给出一个可区分的排列

为了方便切换主题时的间距调整，您可以使用间距调整工具：

- `水平增加/缩减`：调节节点上下间距
- `垂直增加/缩减`：调节节点左右间距

其他：

- `将布局保存为图片`：导出当前画布为 PNG 图片，若有选中节点则仅导出选中的节点，若无选中节点则导出全部节点

## 分享

分享功能位于顶部导航栏，可以帮助您快速分享当前的 Pipeline 配置。

### 使用方式

1. **生成分享链接**：点击顶部导航栏的「分享」按钮
2. **自动复制**：分享链接会自动复制到剪贴板
3. **分享给他人**：将链接发送给其他人
4. **打开分享链接**：接收方打开链接后，会自动创建一个名为「来自分享」的新文件，并加载分享的 Pipeline

分享链接采用 **URL 参数** 方式，适合分享中小型 Pipeline（理论上支持到 1k 个节点，但链接会很长）

::: warning

- 分享链接包含完整的 Pipeline 配置（节点、连接、参数等），请注意隐私安全
- 打开分享链接时会自动创建新文件，不会覆盖现有工作
- 分享的内容不包含本地文件路径等环境相关配置
  :::

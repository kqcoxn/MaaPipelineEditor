## 💡 功能速查

### 节点操作

- **添加节点**：右键空白区域 → 选择模板 → 点击或回车
- **修改节点字段**：选中节点 → 在字段面板中编辑
- **修改节点名**：字段面板中修改 `key` 值
- **复制粘贴**：`Ctrl+C` / `Ctrl+V`
- **删除节点**：`Delete` 或 `Backspace`
- **保存为模板**：字段面板工具栏 → 保存模板按钮

### 连接操作

- **创建连接**：拖拽出口端点到入口端点
- **调整曲率**：悬停连接线 → 拖拽中点手柄
- **删除连接**：选中连接 → `Delete` 或 `Backspace`
- **调整顺序**：点击连接 → 在连接面板中调整

### 文件操作

- **新建文件**：左上角 `+` 按钮
- **导入文件**：Pipeline 面板 → 从粘贴板/文件导入
- **导出文件**：Pipeline 面板 → 部分导出/全部导出
- **保存到本地**：JSON 按钮 → 保存到本地（需本地服务）

### 视图操作

- **缩放**：鼠标滚轮
- **拖动**：按住空白区域拖动
- **字段/连接面板模式**：设置面板切换固定/拖动模式，拖动模式可自由拖动面板位置
- **聚焦透明度**：设置面板启用 → 选中节点自动聚焦
- **路径高亮**：设置面板启用 → 选择起点和终点高亮路径
- **对齐节点**：选中多个节点 → 排版工具
- **自动布局**：未选中节点时 → 排版工具 → 自动布局
- **导出为图片**：排版工具 → 将布局保存为图片

### 快捷键

| 快捷键      | 功能     |
| ----------- | -------- |
| `Ctrl+C`    | 复制节点 |
| `Ctrl+V`    | 粘贴节点 |
| `Ctrl+Z`    | 撤销操作 |
| `Ctrl+Y`    | 重做操作 |
| `Delete`    | 删除     |
| `Backspace` | 删除     |
| `Esc`       | 关闭面板 |
| `Enter`     | 确认选择 |

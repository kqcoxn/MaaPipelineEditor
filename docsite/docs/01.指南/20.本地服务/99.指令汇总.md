# LocalBridge 指令汇总

LocalBridge 的可执行文件名为 `mpelb`，本页汇总了目前所有可用的命令行指令、全局参数以及配置管理命令，方便您在部署和日常使用时快速查阅。

::: tip
命令行参数的优先级高于配置文件。您可以用配置文件保存常用配置，再用命令行参数临时覆盖。
:::

## 命令格式

```bash
mpelb [command] [args] [flags]
```

- 直接执行 `mpelb` 即可启动本地服务。
- `command` 为子命令，如 `config`、`info`。
- `flags` 为全局或命令级参数。

## 全局参数

| 参数 | 简写 | 说明 | 适用场景 |
|------|------|------|----------|
| `--config` | - | 指定自定义配置文件路径 | 启动、config 子命令 |
| `--root` | - | 指定 Pipeline 文件扫描根目录 | 启动 |
| `--port` | - | 指定 WebSocket 服务端口 | 启动 |
| `--log-dir` | - | 指定日志输出目录 | 启动 |
| `--log-level` | - | 指定日志级别：`DEBUG` / `INFO` / `WARN` / `ERROR` | 启动 |
| `--portable` | - | 便携模式：配置与日志存放在可执行文件同目录 | 启动、`config`、`info` |
| `--version` | `-v` | 显示当前版本号 | 任意 |

## 启动服务（根命令）

不加子命令时，`mpelb` 会启动 LocalBridge WebSocket 服务。

### 快速启动

```bash
mpelb
```

### 常用启动示例

```bash
# 指定扫描目录
mpelb --root D:/my-pipelines

# 指定服务端口
mpelb --port 9999

# 调整日志级别
mpelb --log-level DEBUG

# 便携模式启动
mpelb --portable

# 组合使用
mpelb --root ./pipelines --port 9066 --log-level INFO --log-dir ./logs
```

::: warning 注意
修改 `--port` 后，需要在编辑器配置面板中同步修改端口号，否则前端无法正常连接。
:::

## 配置管理命令：`mpelb config`

`config` 是一组用于管理 LocalBridge 配置的子命令。

| 命令 | 参数 | 说明 |
|------|------|------|
| `mpelb config open` | - | 使用系统默认编辑器打开配置文件 |
| `mpelb config set-lib [path]` | `path`（可选） | 设置 MaaFramework lib/bin 目录路径，设置后自动启用 `maafw.enabled` |
| `mpelb config set-resource [path]` | `path`（可选） | 设置 OCR 资源路径（`model/` 所在目录），设置后自动启用 `maafw.enabled` |
| `mpelb config open-log` | - | 使用系统文件管理器打开日志目录 |

### 示例

```bash
# 打开配置文件
mpelb config open

# 设置 MaaFramework 库路径
mpelb config set-lib C:\MaaFramework\bin

# 交互式设置 OCR 资源路径
mpelb config set-resource

# 打开日志目录
mpelb config open-log
```

::: tip
`set-lib` 与 `set-resource` 在设置路径后会自动启用 MaaFramework，无需手动修改 `maafw.enabled`。
:::

## 信息查看命令：`mpelb info`

显示 LocalBridge 当前的运行模式、数据目录、配置文件路径、日志目录和可执行文件路径等信息。

```bash
mpelb info
```

在便携模式下查看：

```bash
mpelb info --portable
```

## 运行模式说明

LocalBridge 会根据启动参数和环境自动选择配置存储位置：

| 模式 | 触发条件 | 配置/日志存储位置 |
|------|----------|-------------------|
| 本地模式 | 默认 | 系统用户数据目录 |
| 便携模式 | 使用 `--portable` | 可执行文件同目录 |
| 开发模式 | 可执行文件旁存在 `config/` 目录 | 可执行文件同目录 |

## 常用组合速查

| 用途 | 命令 |
|------|------|
| 默认启动 | `mpelb` |
| 指定项目目录启动 | `mpelb --root ./Assets` |
| 调试模式启动 | `mpelb --log-level DEBUG` |
| 便携模式启动 | `mpelb --portable` |
| 查看路径信息 | `mpelb info` |
| 打开配置文件 | `mpelb config open` |
| 配置 MaaFramework 库 | `mpelb config set-lib <path>` |
| 配置 OCR 资源 | `mpelb config set-resource <path>` |
| 打开日志目录 | `mpelb config open-log` |
| 查看版本 | `mpelb -v` |

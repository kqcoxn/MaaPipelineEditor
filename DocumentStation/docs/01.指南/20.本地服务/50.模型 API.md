## 概述

MPE 提供模型 API 底座，用于为后续能力统一接入不同服务商。底座只负责配置、请求构造、响应解析和网络代理，不包含任何上层产品功能。

## 配置

在配置面板的「模型 API」区域填写：

- **API 类型**：选择 OpenAI、Anthropic、Gemini 或 OpenAI 兼容服务。
- **API URL**：填写服务基础地址或完整端点。
- **API Key**：密钥会使用 AES-GCM 加密保存在浏览器本地。
- **模型**：填写服务商提供的模型名称。
- **温度**：设置模型生成参数，范围为 0-1。

## LocalBridge 代理

开启「LocalBridge 代理」后，请求会通过本地服务转发，可绕过浏览器 CORS 限制。底座同时支持普通请求和 SSE 流式请求；LocalBridge 未连接时可关闭代理并直接访问支持 CORS 的服务。

底层协议保留 `ai.complete`、`ai.stream.start` 和 `ai.stream.cancel` 请求，以及 `ai.response` 和 `ai.streamChunk` 事件。调用方必须自行传入完整消息数组，底座不会保存会话状态，也不会注入预设提示词。

## 安全提示

- API Key 仅保存在当前浏览器配置中，不会写入项目文件。
- 通过 LocalBridge 代理时，请确认服务只监听受信任的本地地址。
- 发送给外部服务的消息、图片和其他上下文由调用方负责控制。

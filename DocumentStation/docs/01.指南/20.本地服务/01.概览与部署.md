LocalBridge 是 MaaPipelineEditor 唯一的业务后端，提供工作区文件、资源、MaaFramework、调试、工具和模型 API 代理能力。当前实现基于 Python、FastAPI 与官方 `MaaFw` Python Binding，通信协议固定为 `2.3.0`。

纯 Web 编辑不依赖 LocalBridge；本地文件管理、设备连接、截图识别和流程调试需要 LocalBridge 或 Desktop。

## 环境要求

- 同架构 CPython 3.11-3.14
- Windows x64、macOS Apple Silicon 或 Linux x64
- 首次安装可访问 PyPI

`MaaPipelineEditor-LocalBridge` 会依赖精确版本的 `MaaFw`，MaaFramework native library 随 Python wheel 安装，不需要另行下载 DLL、so 或 dylib。原生 OCR 模型仍需由具体资源项目提供，并通过 `maafw.resource_dir` 配置。

## 安装

推荐使用 `uv tool` 创建隔离环境：

```bash
uv tool install MaaPipelineEditor-LocalBridge
```

也可以在自行创建的 venv 中安装：

```bash
python -m venv .venv
# 激活 .venv 后执行
python -m pip install MaaPipelineEditor-LocalBridge
```

升级时执行：

```bash
uv tool upgrade MaaPipelineEditor-LocalBridge
```

## 环境检查

```bash
mpelb version
mpelb doctor --json
```

`doctor` 会报告 LocalBridge、协议、Python 与 MaaFw 版本，以及固定图片控制器当前使用 `native-dbg` 还是 `custom-fallback`。

## 启动与连接

在 Pipeline 工作区中运行：

```bash
mpelb serve --root ./Assets
```

LocalBridge 仅监听 `127.0.0.1`，并用 URL fragment 将实际端口交给稳定版 Editor；Editor 将端口写入当前标签页的 `sessionStorage` 后立即清除 fragment。

常用参数：

```bash
mpelb serve --root ./Assets --port 9066
mpelb serve --root ./Assets --no-open
mpelb serve --config ./localbridge.json
mpelb serve --allow-origin https://editor.example.com
```

自部署 Editor 的 Origin 不在默认允许列表中，必须通过 `--allow-origin` 或 `server.allowed_origins` 显式添加。

## 配置工作区与 OCR 资源

```bash
mpelb config set file.root '"D:/Project/Assets"'
mpelb config set maafw.resource_dir '"D:/Project/resource"'
mpelb config show
```

配置写入采用原子替换。文件 API 只接受工作区相对路径，资源目录只作为显式配置的只读允许根。

## 运行行为

- WebSocket 地址为 `/v2/ws`，首帧必须执行 `system.hello`。
- 连接中断后 Editor 保持断开，不会自动重试；用户手动连接后按调试事件序号续传。
- 图片通过 `/v2/artifacts` 二进制接口传输，不进入 WebSocket JSON。
- 调试运行使用当前未保存内容生成不可变内存快照，不会自动覆盖源文件。
- 首版不支持真正的暂停、继续或断点。

完整参数见 [指令汇总](/guide/server/commands)，配置结构见 [进阶配置](/guide/server/advance)。

## 常见问题

### `doctor` 失败

确认当前终端使用 CPython 3.11-3.14，并重新安装 LocalBridge。不要把其他 MaaFw native library 手动复制进该环境。

### Editor 无法连接

请从本次 `mpelb serve` 输出或自动打开的 URL 进入 Editor。使用随机端口时，LocalBridge 重启后需要使用新打开的页面。

### OCR 不可用

`MaaFw` wheel 包含运行库，但不包含业务资源的 OCR 模型。将包含 `model/ocr` 的资源根目录配置到 `maafw.resource_dir`。
